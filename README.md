# Rent Calculator (Python)

This is a simple Python program to calculate the monthly share of expenses per person in a shared flat. 

## Features
- Takes user input for rent, food expenses, electricity usage, charge per unit, and number of residents.
- Calculates total electricity bill.
- Splits total expenses evenly among all residents.
- Provides clear, CLI-based output.

## Skills Demonstrated
- Python basics: variables, input, arithmetic operations
- Problem-solving and logic implementation
- Command-line interface (CLI) programming

## How to Use
1. Run `rent_calculator.py`
2. Enter the required inputs when prompted
3. Get the per-person expense
